Trema

A platform for developing OpenFlow controllers

Download & Install

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/trema/trema

Just run build.rb and straight away you have a working Trema installation. 

$ ./build.rb

Get Started

To start using Trema, cbench is a good entry point. cbench is a program for testing OpenFlow controllers by generating packet-in events for new flows. To estimate the performance of Trema sample controller with cbench, run the following command. 
$ ./build.rb cbench
More samples including openflow-event dumper, learning-switch controller and routing-switch controller etc... are found in quick-start guide and [trema]/src/examples.
